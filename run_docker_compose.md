#Run docker compose 

For the start the docker or multiple docker using docker compose is need the create the file 'docker-compose.yml'.
in this file exit the config the multiple dockers and servises. using this file we can config the networks , ports ,
conteiners and the start the project.

utilis comands:

```bash
docker compose up -> start the docker usint the docker compose
docker compose up --build -> build the all dockers and start the conteiner 
docker exec -it <docker_name> <comand> -> this command execute the command in docker since this is a run.
```
